Save your extensions in the 'plugins' folder.

Plugin folder can be put anywhere in the file system.

Path to folder must be specified in SOFE.getPluginPath()

A plugin can be loaded by typing ">> SOFE.plugin('myplugin')"

A plugin can be unloaded by typing ">> SOFE.plugout('myplugin')"

Loaded plugins can be listed by typing ">> SOFE.pluggedin()"
